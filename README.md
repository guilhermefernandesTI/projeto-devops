# 🚀 Projeto DevOps

Este repositório foi criado como parte de uma atividade acadêmica com o objetivo de demonstrar o uso do GitHub e práticas de controle de versão no contexto de DevOps.

---

## 📌 Objetivo

Aplicar conceitos de:
- Controle de versão
- Branches e merges
- Versionamento de código
- Organização de projetos no GitHub

- Linguagem: Java (exemplo)

- GitHub Desktop
